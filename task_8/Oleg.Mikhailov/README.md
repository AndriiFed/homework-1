1. В task1 объект Flashlight имеет свойства: 
- Работа в трёх режимах (белый свет при 1 батарейке, красный свет при 2 и более батарейках, мигающий режим при 3 и более батарейках)
Все режимы можно устанавливать в любой момент работы фонарика, главное необходимо иметь достаточное кол-во батареек.
- Максимальное кол-во батареек - 3 шт.

2.1. В task2 объект AutoDoors  - система автоматического подсчёта людей в помещении, а так же автоматического открытия дверей.
2.2. В task2 объект Signaling - система безопасности, которая откроется только при корректном вводе пароля. 
Для ввода есть всего 3 попытки, после чего система будет заблокирована и необходимо ожидать звонка от пульта управления для подтверждения личности.
