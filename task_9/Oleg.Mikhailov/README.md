1. В **merge** поступает два массива, которые потом оба добавляются в массив result. 
Массив result сортируется по возрастанию.

2. Сложность  **Algorithm** - это параболлическая функция, отображает правую ветвь параболлы. График отображён на скриншоте. 
Уравнение данной функции было получено путём аппроксимации функции исходя из полученных координат и имеет следующий вид - `O(0,16n^3 - 0,5n^2 + 0,3n - 7,5*10^(-14))`

3. В **sumExists** передаётся массив и число, с которым будет проводиться сравнение. 
Если сумма двух каких-то элементов массива совпадает со значением sum, то функция возвращает true.
